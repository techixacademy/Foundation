This foundation repo outlines the list of foundation materials to go through before starting the journey of any defined piller.

Foundation Data:
1. The user is aware about the key operating system like Windows, Linux, VMware
2. Hardware Architecture: Microcontroller, Intel, AMD, ARM
3. Programming Languages: Low level language - C/C++, Scripting Language: Python, Go
4. AI Tools: Chatgpt, Lovable, ElevanLabs, Google Gemini, Claude, Microsoft Copilot
5. Docker, Container, Kubernetes
6. Waterfall, Agile, ScaledAgile
7. SDLC
8. On Prem/In Prem, Cloud Platforms: AWS, Azure, GCP, Railway


Pillar	                Technical Prerequisites

P1: HW/IoT/EV	        Basic Electronics (Circuit laws), C/C++ basics, Embedded logic.	
P2: Cloud Native	    Linux CLI, Basic Networking, Virtualization concepts.	
P3: AI Full Stack	    Proficiency in Python, HTML/CSS, Web APIs (REST).	
P4: AI Models/Ops	    Python (Advanced), Docker basics, Data handling.	
P5: Cyber Security	    OS Fundamentals (Windows/Linux), Networking protocols.	
P6: Marketing/Digital	Basic Excel/Google Sheets, Writing/Communication,Statistics, Data Visualization, Canva, Adobe
